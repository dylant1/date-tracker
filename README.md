# Date Tracker

Need an easy way to manage upcoming events and important dates?


## Hosting Locally
*Note: Make sure docker and docker-compose are installed*

**1. Clone the repository**

**2. From the CLI:**
```bash
cd backend
docker-compose up -d
cd ../frontend
npm run dev
```

**3. Open** ``http://localhost:3000/``
